## Usage

### Requirements

- [Node](https://nodejs.org) >= 12
- [yarn](https://yarnpkg.com) v1

### How to start

- Clone the repository
- Run `yarn install`
- Run `yarn start`
- Open [http://localhost:3000](http://localhost:3000) in browser

## Test

- Run `yarn test`
- For coverage, run `yarn test:cover`
